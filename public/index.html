📱 Cómo integrar en tu APK Android
1. Verificar actualizaciones:
Al abrir la app, llama a GET /ultragol1/version. Si el versionCode del servidor es mayor al tuyo → muestra diálogo de actualización.

2. Mostrar notificaciones:
Llama a GET /notifications. El JSON devuelve un array con title, message y type.

3. Descargar APK:
El campo downloadUrl en la respuesta de versión ya contiene el enlace directo.
